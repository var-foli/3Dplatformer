# 3Dplatformer

A 3D platformer game inspired by Super Mario 64, built with Unity.

### Gameplay
- **Platforming Mechanics**: Walk, run, and jump to navigate through 3D environments.
- **Enemies**: Take damage when running into enemies, and destroy enemies by jumping on them.
- **Collectibles**: Collect coins and stars to gain points and win levels.

### Installation
Prerequisites: Unity 6000.3.2f1 or higher

```git clone https://github.com/var-foli/3Dplatformer.git```
